# docs/landing

Static landing page for TunnelBox.

| File | Purpose |
|---|---|
| `index.html` | Main landing page — hero, Free vs Pro comparison table, use-case cards, FAQ, footer. Single-file HTML with inline CSS. No build step required. |

## Hosting

Host the `docs/landing/` directory as a static site (e.g., GitHub Pages, Cloudflare Pages, Netlify).
The page is designed to live at `tunnelbox.app/` or `tunnelbox.app/pricing`.

## Content implemented

- ES-113 (Story 113): Hero, price cards, 30-day refund, EULA/refund-policy footer links, Cloudflare attribution
- ES-224 (Story 224): Free vs Pro comparison table, use-case cards, FAQ including "What's NOT in Pro", lifetime/refund fine print
