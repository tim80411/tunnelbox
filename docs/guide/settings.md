# Settings

點擊右上角齒輪圖示或按 `⌘,`（macOS）/ `Ctrl+,`（Windows / Linux）開啟設定面板。

## 偏好設定

| 設定項目 | 說明 | 預設值 |
|---------|------|--------|
| Auto-start servers | 啟動應用程式時自動啟動所有伺服器 | 關閉 |
| Default serve mode | 新增站點時的預設模式（Static / Proxy） | Static |

## 版本與更新

設定面板底部顯示目前版本號。點擊 **檢查更新** 按鈕即可手動檢查是否有新版本可用。

更新流程：

1. 點擊 **檢查更新**
2. 若有新版本，會顯示版本號及下載按鈕
3. 下載完成後，重新啟動即可套用更新

## 鍵盤快捷鍵

按 `⌘/`（macOS）/ `Ctrl+/`（Windows / Linux）開啟快捷鍵面板。

### General

| 快捷鍵 | 動作 |
|--------|------|
| `⌘N` / `Ctrl+N` | 新增站點 |
| `⌘,` / `Ctrl+,` | 開啟設定 |
| `⌘/` / `Ctrl+/` | 開啟快捷鍵面板 |
| `Esc` | 關閉面板 / 彈窗 |

### Sites

| 快捷鍵 | 動作 |
|--------|------|
| `↑` / `↓` | 上下切換站點 |
| `⌘O` / `Ctrl+O` | 在瀏覽器中開啟 |
| `⌘R` / `Ctrl+R` | 重啟伺服器 |
| `⌘⌫` / `Ctrl+Backspace` | 刪除站點 |
| `⌘V` / `Ctrl+V` | 貼上路徑新增站點 |

### Window（macOS）

| 快捷鍵 | 動作 |
|--------|------|
| `⌘W` | 關閉視窗 |
| `⌘M` | 最小化 |
| `⌘H` | 隱藏 TunnelBox |
| `⌘Q` | 退出 |
