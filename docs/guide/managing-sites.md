# Managing Sites

## 新增網頁

1. 點擊右上角 **+ Add Site**
2. 輸入名稱（或留空，會自動使用資料夾名稱）
3. 點擊 **Browse** 選擇包含靜態網頁的資料夾
4. 點擊 **Confirm**

TunnelBox 會自動啟動本地伺服器，分配一個可用的 Port（範圍 3000–9000）。

## 網頁清單

每個網頁項目顯示：

| 欄位 | 說明 |
|------|------|
| 名稱 | 你設定的網頁名稱 |
| 路徑 | 本地資料夾路徑 |
| URL | `http://localhost:PORT`，可點擊複製 |
| 狀態 | 運行中 / 已停止 / 錯誤 |

## 操作

- **Open** — 在瀏覽器中開啟（若已停止，會自動啟動後開啟）
- **Start / Stop** — 手動啟動或停止本地伺服器
- **Remove** — 刪除網頁（停止伺服器，但不刪除本地檔案）

## Hot Reload

當你修改資料夾內的檔案時，已在瀏覽器中開啟的頁面會自動重新載入，不需要手動重新整理。

## 持久化

所有網頁設定會自動儲存。關閉 TunnelBox 後重新開啟，之前的網頁會自動恢復並啟動伺服器。
