# v1.2.0

**Release Date:** 2026-03-21

## Highlights

TunnelBox v1.2.0 帶來多項重大功能更新，包括反向代理模式、系統匣常駐、LAN 分享、自動更新等，大幅提升使用體驗。

## New Features

### Proxy 模式
- 新增反向代理模式，可將已運行的開發伺服器（如 Vite、Next.js）透過 Tunnel 對外分享
- 支援 HTTP 和 WebSocket 轉發
- 目標伺服器無回應時顯示 502 Bad Gateway 提示頁面

### System Tray
- 應用程式常駐於系統匣，關閉視窗不會結束程式
- 系統匣選單顯示所有站點及其狀態
- macOS 支援 Template Image 適應亮色 / 暗色模式

### LAN 分享
- 站點運行中自動偵測區域網路 IP 並顯示 LAN URL
- LAN URL 附有「Local」標籤便於識別
- QR Code 彈窗支援手機掃碼快速預覽
- 自動過濾 VPN 介面，優先顯示主要網路

### 自動更新
- 設定面板中新增版本檢查功能
- 支援自動下載及安裝更新
- 強制更新機制確保不會使用過舊版本

### 設定面板
- Auto-start servers：啟動應用程式時自動啟動所有伺服器
- Default serve mode：設定新增站點時的預設模式（Static / Proxy）

### 鍵盤快捷鍵
- 完整快捷鍵支援（`⌘N` 新增、`⌘O` 開啟、`⌘R` 重啟、`↑/↓` 切換站點等）
- `⌘/` 開啟快捷鍵面板查看所有快捷鍵

### 拖放與剪貼簿
- 拖曳資料夾到視窗快速新增站點
- 剪貼簿貼上路徑（`⌘V` / `Ctrl+V`）快速新增站點
- 支援從 Finder 複製資料夾後直接貼上

### CLI
- 新增命令列介面 `tunnelbox`
- 支援站點管理（`site list` / `site add` / `site remove`）
- 支援伺服器控制（`server start` / `server stop`）
- 支援 Tunnel 操作（`tunnel start` / `tunnel stop`）
- 支援環境檢查（`env check`）
- 支援 JSON 輸出格式（`--json`）

### Linux 支援
- 新增 Linux 平台打包（`.deb` / `.rpm` / `.AppImage`）
- 支援 cloudflared 自動偵測與安裝

## Improvements

- URL 顯示重構：長 URL 自動截斷顯示，hover 時顯示完整 URL
- 複製按鈕反饋優化，複製成功後顯示確認提示
- QR Code 改為彈窗顯示（取代 popover），提升易用性
- CSS tooltip 取代 JavaScript tooltip，改善效能

## Platforms

- macOS (Universal — Intel + Apple Silicon) `.dmg`
- Windows (x64) `.exe`
- Linux (x64) `.deb` / `.rpm` / `.AppImage`
