# v1.0.0

**Release Date:** 2026-03-18

## Highlights

TunnelBox 首次發布！一款桌面應用程式，讓你管理本地靜態網頁並透過 Cloudflare Tunnel 分享給任何人。

## Features

### Local Server Management
- 選擇本地資料夾，自動啟動靜態伺服器
- 自動分配可用 Port（3000–9000）
- 檔案變更時自動 Hot Reload
- 持久化儲存，重啟後自動恢復

### Quick Tunnel
- 一鍵產生臨時公開 URL（無需帳號）
- 公開 URL 一鍵複製
- 停止伺服器時自動關閉 Tunnel
- 斷線自動重連（最多 3 次）

### Named Tunnel
- 登入 Cloudflare 後建立持久公開 URL
- 重啟應用後自動恢復
- 啟動 / 停止 / 刪除管理
- 斷線自動重連

### Custom Domain
- 將自訂網域綁定到 Named Tunnel
- 自動建立 DNS CNAME 記錄
- 綁定狀態顯示（傳播中 / 已綁定）

### Cloudflare Auth
- OAuth 登入（透過瀏覽器授權）
- 認證持久化（重啟不需重新登入）
- 登出時自動停止 Named Tunnel

## Platforms
