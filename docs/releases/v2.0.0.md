# TunnelBox v2.0.0

> 自 v1.4.0 以來累積的重大版本：Pro 付費方案正式上線、全新介面、資安全面加固，並將執行環境升級到 Electron 42。

## 重點

- 🎉 **Pro 付費方案上線** — 授權匯入、分級功能、跨裝置授權
- 🎨 **全新 Direction B 介面** — 主從（master-detail）版面與重新設計的操作流程
- 🔒 **資安全面加固** — 導航鎖、CSP、SSRF 防護、Host 校驗等 20+ 項
- ⬆️ **Electron 33 → 42**（Chromium 148 / Node 24）

## ⚠️ 升級須知（重大變更）

| 項目 | 變更 |
|------|------|
| 最低系統需求 | **macOS 12 (Monterey) 以上**。macOS 11 (Big Sur) 不再支援，Big Sur 使用者將停止收到自動更新。 |
| 執行環境 | 升級到 Electron 42（Chromium 148 / Node 24） |
| macOS 安裝檔 | 本版**未經 Apple 簽章／公證**；首次開啟時 Gatekeeper 會提示，請於「系統設定 → 隱私權與安全性」允許開啟。 |

## 新功能

### Pro 付費方案

| 功能 | 說明 |
|------|------|
| 授權匯入 | 拖放授權檔、Activate Pro 選擇器、自動掃描 Downloads 目錄 |
| 分級（Community / Pro） | 以 Ed25519 簽章驗證的 tier-gate |
| 使用限制引擎 | 並發分享數限制、多 Cloudflare 帳號、背景常駐檢查 |
| 跨裝置授權 | 不計啟用次數、不回傳（no phone-home） |
| 續約提醒 | soft-lock banner，每個 major.minor 只提示一次 |
| 官網 | Pro 介紹頁與退款政策頁 |

### 全新介面（Direction B）

- 主從版面：左側站點列（SiteRail）+ 右側詳情（SiteDetail）
- 重新設計：新增站點 sheet、系統設定 modal、通知 popover、分享紀錄卡片
- 標籤 chips、每列快速開啟／複製、側邊 Share CTA

## 安全性

- 導航鎖 + `openExternal` scheme 白名單
- renderer 注入 Content-Security-Policy
- proxy 目標 SSRF 分類 + 分享內部／metadata 目標前的確認對話框
- WebSocket 升級與 proxy 路徑的 Host/Origin 反 DNS-rebinding 校驗
- 全域連線數上限；本地 API server 不再信任 `X-Forwarded-For`
- 靜態伺服器封鎖敏感目錄與 dotfiles
- 自訂網域於傳入 cloudflared 前驗證；`frpc.toml` 值 escape；bore `serverAddr` 驗證
- 相依套件 CVE 全數升級（`pnpm audit --prod` 乾淨）

## 效能與穩定性

- tray-only renderer 休眠模式（縮小待機記憶體）
- 檔案 watcher 健康監測 + ignore 清單
- cloudflared tunnel watchdog + 錯誤翻譯層
- LAN 模式 per-site 開關 + 條件式 bind（預設僅 127.0.0.1）

## 平台與工具鏈

- Electron 42.5.0、electron-builder 26.15.3、electron-vite 3.1.0
- electron-updater 維持 6.8.x
